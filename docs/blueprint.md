# **App Name**: LinguaLeap

## Core Features:

- Lesson Delivery: Deliver bite-sized language lessons in single-unit steps, each lesson capped at 3-5 minutes, and include a progress bar/timer to indicate the current status of the lesson. Serves grouped flashcards and dialogue lines in these single unit steps. Presents a quiz (MCQ/fill-in-the-blank/audio-based) upon lesson completion.
- Flashcard Management: A personal flashcard interface, per-item spaced repetition using FSRS-lite or SM-2.
- Audio Comparison Tool: Each flashcard/dialogue displays reference audio and supports audio recording/comparison by user. The app presents the recording as a tool the user can use to evaluate pronunciation..
- Smart Scheduler: AI-powered scheduler that tracks user performance and predicts optimal intervals for flashcard reviews using FSRS-lite or SM-2. This allows personalized study plans and improves learning outcomes. The LLM uses the intervals provided by the tool, combined with historical information about the user's usage patterns, to adjust the reminders to maximize their likelihood of engaging with the app..
- Settings and Preferences: Allow users to customize language preferences.

## Style Guidelines:

- Primary color: Vibrant blue (#29ABE2) to evoke a sense of trust and intellectual stimulation.
- Background color: Light gray (#F0F0F0), for a clean and uncluttered look.
- Accent color: Yellow-orange (#F2994A) to highlight key interactive elements, creating a cheerful feeling.
- Headline font: 'Poppins', sans-serif, to give a contemporary and precise feel to headings. Use 'PT Sans' for body text.
