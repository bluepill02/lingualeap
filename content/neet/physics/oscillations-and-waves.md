# NEET Physics: Oscillations and Waves - அலைவுகள் மற்றும் அலைகள்

## Concept Notes (கருத்து குறிப்புகள்)

### 1. Oscillations (அலைவுகள்)

ஒரு பொருள் ஒரு சமநிலை புள்ளியை (equilibrium position) சுற்றி முன்னும் பின்னுமாக நகரும் இயக்கம் அலைவு இயக்கம் எனப்படும். இது ஒரு குறிப்பிட்ட கால இடைவெளியில் மீண்டும் மீண்டும் நிகழும் இயக்கம்.

**எடுத்துக்காட்டு:** கடிகாரத்தின் ஊசல் (pendulum of a clock), ஸ்பிரிங்கில் தொங்கும் நிறை (mass on a spring).

**தொடர்புடைய தமிழ்த் தொடர்பு:** கோவில்களில் தொங்கும் மணி (temple bell) அசைவது அல்லது சிறுவர்கள் ஆடும் ஊஞ்சல் (children's swing) ஒரு நல்ல அலைவு இயக்கத்திற்கான எடுத்துக்காட்டுகள்.

#### Simple Harmonic Motion (SHM) - எளிய சீரிசை இயக்கம்

எளிய சீரிசை இயக்கம் என்பது ஒரு சிறப்பு வகை அலைவு இயக்கம். இதில், ஒரு பொருளின் மீது செயல்படும் மீள்விசை (restoring force) அதன் சமநிலை புள்ளியிலிருந்து ஏற்படும் இடப்பெயர்ச்சிக்கு (displacement) நேர் விகிதத்தில் இருக்கும் மற்றும் எப்போதும் சமநிலை புள்ளியை நோக்கி செயல்படும்.
$F \propto -x \implies F = -kx$ (ஹூக் விதி - Hooke's Law)

**முக்கிய வரையறைகள் (Key Definitions):**

1.  **இடப்பெயர்ச்சி (Displacement, x):** சமநிலை புள்ளியிலிருந்து பொருளின் தூரம்.
2.  **வீச்சு (Amplitude, A):** சமநிலை புள்ளியிலிருந்து பொருளின் அதிகபட்ச இடப்பெயர்ச்சி. (ஊஞ்சல் எவ்வளவு தூரம் முன்னும் பின்னும் செல்கிறது)
3.  **அலைவுக்காலம் (Time Period, T):** ஒரு முழு அலைவுக்கு (ஒரு முழு சுழற்சிக்கு) ஆகும் நேரம். அலகு: வினாடி (s). (ஒரு ஊஞ்சல் ஒருமுறை சென்று திரும்பி வர ஆகும் நேரம்)
4.  **அதிர்வெண் (Frequency, f or ν):** ஒரு வினாடியில் நிகழும் அலைவுகளின் எண்ணிக்கை. $f = 1/T$. அலகு: ஹெர்ட்ஸ் (Hz).
5.  **கோண அதிர்வெண் (Angular Frequency, ω):** $ω = 2πf = 2π/T$. அலகு: ரேடியன்/வினாடி (rad/s).
6.  **கட்டம் (Phase, φ):** அலைவு இயக்கத்தின் எந்த ஒரு கணத்திலும் பொருளின் நிலை மற்றும் திசையை குறிக்கும்.

**SHM-ன் சமன்பாடுகள் (Equations of SHM):**

*   **இடப்பெயர்ச்சி (Displacement):** $x(t) = A \sin(\omega t + \phi)$
*   **திசைவேகம் (Velocity):** $v(t) = \frac{dx}{dt} = A\omega \cos(\omega t + \phi) = \pm \omega \sqrt{A^2 - x^2}$
*   **முடுக்கம் (Acceleration):** $a(t) = \frac{dv}{dt} = -A\omega^2 \sin(\omega t + \phi) = -\omega^2 x$

**SHM-ல் ஆற்றல் (Energy in SHM):**

*   **நிலையாற்றல் (Potential Energy, PE):** $PE = \frac{1}{2} k x^2 = \frac{1}{2} m \omega^2 x^2$
*   **இயக்க ஆற்றல் (Kinetic Energy, KE):** $KE = \frac{1}{2} m v^2 = \frac{1}{2} m \omega^2 (A^2 - x^2)$
*   **மொத்த ஆற்றல் (Total Energy, E):** $E = PE + KE = \frac{1}{2} m \omega^2 A^2 = \frac{1}{2} k A^2$
    *   SHM-ல் மொத்த ஆற்றல் மாறிலியாக இருக்கும்.

**SHM-ன் சில அமைப்புகள் (Examples of SHM Systems):**

1.  **ஸ்பிரிங்-நிறை அமைப்பு (Spring-Mass System):**
    *   அதிர்வெண்: $f = \frac{1}{2\pi} \sqrt{\frac{k}{m}}$
    *   அலைவுக்காலம்: $T = 2\pi \sqrt{\frac{m}{k}}$
    *   இங்கு $k$ என்பது ஸ்பிரிங் மாறிலி (spring constant), $m$ என்பது நிறையின் அளவு (mass).

    ```
    -----| ஸ்பிரிங் |-----
        |         |
        |         |
        |  m (நிறை)  |
        |         |
        -----------
          (மேசை/நிலம்)
    ```
    *ஸ்பிரிங்கில் தொங்கும் நிறை - Spring-Mass System*

2.  **எளிய ஊசல் (Simple Pendulum):**
    *   அலைவுக்காலம்: $T = 2\pi \sqrt{\frac{L}{g}}$ (சிறிய கோண அலைவுகளுக்கு மட்டும்)
    *   இங்கு $L$ என்பது ஊசல் கயிற்றின் நீளம் (length of the pendulum), $g$ என்பது புவி ஈர்ப்பு முடுக்கம் (acceleration due to gravity).

    ```
          O (தாங்கும் புள்ளி)
          |
          | L (நீளம்)
          |
          v
         (கோளம் - நிறை)
           m
    ```
    *எளிய ஊசல் - Simple Pendulum*
    **தமிழ்த் தொடர்பு:** பழைய காலத்தில் வீடுகளில் குழந்தைகள் ஆடும் ஊஞ்சல் (swing) அல்லது கோவில்களில் பெரிய மணி (bell) ஆடுவது.

#### Damped Oscillations (தணிந்த அலைவுகள்)

ஒரு அலைவுறும் அமைப்பின் ஆற்றல் வெளிப்புற எதிர்ப்புகளால் (காற்றின் உராய்வு, பசைத்தன்மை) படிப்படியாக குறைந்து, வீச்சு காலப்போக்கில் குறையுமேயானால், அது தணிந்த அலைவுகள் எனப்படும்.

#### Forced Oscillations and Resonance (கட்டாய அலைவுகள் மற்றும் ஒத்ததிர்வு)

*   **கட்டாய அலைவுகள்:** ஒரு அலைவுறும் அமைப்பின் மீது ஒரு வெளிப்புற, காலமுறை விசை (periodic force) செயல்பட்டு அதனை அலைவுற வைத்தால், அவை கட்டாய அலைவுகள் எனப்படும்.
*   **ஒத்ததிர்வு:** வெளிப்புற விசையின் அதிர்வெண் அமைப்பின் இயற்கையான அதிர்வெண்ணுக்கு (natural frequency) சமமாக இருக்கும்போது, அமைப்பின் வீச்சு மிக அதிகமாக மாறும். இதுவே ஒத்ததிர்வு எனப்படும்.

    **தமிழ்த் தொடர்பு:** ஒரு பாலம் படைவீரர்களின் சீரான நடையால் அதிர்ந்து இடிந்து விழுந்தது (Meldrum Bridge incident) ஒத்ததிர்வுக்கு ஒரு எடுத்துக்காட்டு. ரேடியோ அலைவரிசையை (frequency) சரிசெய்வது ஒத்ததிர்வு கொள்கையின் அடிப்படையிலானது.

### 2. Waves (அலைகள்)

அலை என்பது ஒரு ஊடகத்தில் ஆற்றலை ஒரு இடத்திலிருந்து மற்றொரு இடத்திற்கு மாற்றும் ஒரு தொந்தரவு (disturbance). இதில் ஊடகத்தின் துகள்கள் (particles) தங்கள் சமநிலை புள்ளிகளைச் சுற்றி அலைவுறும், ஆனால் அவை இடமாற்றம் செய்யப்படாது.

**தமிழ்த் தொடர்பு:** குளத்தில் கல் எறியும்போது உருவாகும் அலைகள் (ripples in a pond), அல்லது கடலில் தோன்றும் அலைகள் (ocean waves). இங்கு நீர் அலைவுறும், ஆனால் ஒரு இடத்திலிருந்து மற்றொரு இடத்திற்கு நகராது (பாட்டில்கள் மிதக்கும்).

#### அலைகளின் வகைகள் (Types of Waves):

1.  **குறுக்கலைகள் (Transverse Waves):** ஊடகத்தின் துகள்கள் அலை பரவும் திசைக்கு செங்குத்தாக அலைவுறும்.
    *   **எடுத்துக்காட்டு:** நீரில் ஏற்படும் அலைகள், ஒரு கயிற்றில் ஏற்படும் அலைகள், ஒளி அலைகள்.
    ```
            /\        /\
           /  \      /  \
          /    \    /    \
         /      \/      \
    ----/--------X--------\---------> அலை பரவும் திசை
        \      /\      /
         \    /  \    /
          \  /    \  /
           \/      \/
    <---- துகள்கள் அலைவுறும் திசை
    ```
    *குறுக்கலை - Transverse Wave*

2.  **நெட்டலைகள் (Longitudinal Waves):** ஊடகத்தின் துகள்கள் அலை பரவும் திசைக்கு இணையாக அலைவுறும்.
    *   **எடுத்துக்காட்டு:** ஒலி அலைகள் (sound waves).
    ```
    (C = சுருக்கம், R = நீட்சி)

    R  C   R   C    R   C    R   C
    ||||||| | ||||||| | ||||||| | ||||||| --------> அலை பரவும் திசை
    <---- துகள்கள் அலைவுறும் திசை
    ```
    *நெட்டலை - Longitudinal Wave*

#### அலைகளின் பண்புகள் (Characteristics of Waves):

1.  **வீச்சு (Amplitude, A):** சமநிலை புள்ளியிலிருந்து ஊடக துகள்களின் அதிகபட்ச இடப்பெயர்ச்சி.
2.  **அலைநீளம் (Wavelength, λ):** ஒரு முழு அலைவு சுழற்சியில் அலை பரவும் தூரம். (இரண்டு அடுத்தடுத்த முகடுகளுக்கு (crests) அல்லது பள்ளங்களுக்கு (troughs) இடையேயுள்ள தூரம்).
3.  **அதிர்வெண் (Frequency, f or ν):** ஒரு வினாடியில் ஒரு புள்ளியைக் கடக்கும் முழு அலைவுகளின் எண்ணிக்கை.
4.  **அலைவுக்காலம் (Time Period, T):** ஒரு அலைநீள தூரத்தை கடக்க அலைக்கு ஆகும் நேரம். $T = 1/f$.
5.  **அலை வேகம் (Wave Speed, v):** $v = f\lambda$.

**அலை சமன்பாடு (Wave Equation):**
$y(x,t) = A \sin(kx - \omega t + \phi)$
இங்கு, $k = 2\pi/\lambda$ (அலை எண் - wave number), $ω = 2\pi f$ (கோண அதிர்வெண் - angular frequency).

**பல்வேறு ஊடகங்களில் அலை வேகம் (Wave Speed in Different Media):**

*   **கயிற்றில் குறுக்கலை வேகம் (Transverse wave on a string):** $v = \sqrt{\frac{T}{\mu}}$
    *   இங்கு $T$ என்பது கயிற்றின் இழுவிசை (tension), $μ$ என்பது ஓரலகு நீளத்திற்கான நிறை (linear mass density).
*   **மீள் ஊடகத்தில் நெட்டலை வேகம் (Longitudinal wave in an elastic medium):** $v = \sqrt{\frac{B}{\rho}}$ (திரவங்கள்), $v = \sqrt{\frac{Y}{\rho}}$ (திடப்பொருட்கள்)
    *   இங்கு $B$ என்பது பரும மீட்சிக் குணகம் (bulk modulus), $Y$ என்பது யங்ஸ் குணகம் (Young's modulus), $ρ$ என்பது அடர்த்தி (density).
*   **காற்றில் ஒலி வேகம் (Speed of sound in air):** $v = \sqrt{\frac{\gamma P}{\rho}}$ (லாப்லாஸ் திருத்தம் - Laplace's correction)
    *   இங்கு $γ$ என்பது வெப்ப மாற்றி விகிதம் (adiabatic index), $P$ என்பது அழுத்தம் (pressure), $ρ$ என்பது அடர்த்தி.

#### அலைகளின் மேற்பொருந்துதல் (Superposition of Waves):

இரண்டு அல்லது அதற்கு மேற்பட்ட அலைகள் ஒரு புள்ளியில் ஒரே நேரத்தில் சந்திக்கும்போது, அப்பகுதியிலுள்ள ஊடக துகளின் மொத்த இடப்பெயர்ச்சி தனிப்பட்ட அலைகளின் இடப்பெயர்ச்சிகளின் வெக்டர் கூட்டுத்தொகைக்கு சமம்.

1.  **குறுக்கீட்டு விளைவு (Interference):** இரண்டு ஒத்த அலைகள் (coherent waves) மேற்பொருந்தும்போது ஆற்றலின் மறுபகிர்வு ஏற்படுவது.
    *   **ஆக்கபூர்வ குறுக்கீடு (Constructive Interference):** இரண்டு அலைகளும் ஒரே கட்டத்தில் (in phase) சந்திக்கும்போது, வீச்சு அதிகமாகிறது.
    *   **அழிப்பபூர்வ குறுக்கீடு (Destructive Interference):** இரண்டு அலைகளும் எதிரெதிர் கட்டத்தில் (out of phase) சந்திக்கும்போது, வீச்சு குறைகிறது.

2.  **விம்மல்கள் (Beats):** சற்றே வேறுபட்ட அதிர்வெண் கொண்ட இரண்டு அலைகள் மேற்பொருந்தும்போது, ஒலி செறிவில் (loudness) ஏற்படும் காலமுறை ஏற்ற இறக்கங்களே விம்மல்கள் எனப்படும்.
    *   விம்மல் அதிர்வெண் $f_{beat} = |f_1 - f_2|$.
    **தமிழ்த் தொடர்பு:** இரண்டு புல்லாங்குழல்கள் (flutes) ஒரே நேரத்தில் வாசிக்கப்படும்போது, சற்றே வேறுபட்ட சுருதிகளால் ஏற்படும் 'வூ.. வூ.. வூ..' என்ற ஏற்ற இறக்கம்.

3.  **நிலை அலைகள் (Standing Waves):** ஒரே வீச்சு மற்றும் அதிர்வெண் கொண்ட இரண்டு அலைகள் எதிரெதிர் திசைகளில் பயணித்து மேற்பொருந்தும்போது, ஊடகத்தில் நிலையாகத் தோன்றும் அலைகள் நிலை அலைகள் எனப்படும்.
    *   **கணுக்கள் (Nodes):** குறைந்தபட்ச வீச்சு (அதிகபட்சமாக பூஜ்ஜியம்) கொண்ட புள்ளிகள்.
    *   **எதிர்கணுக்கள் (Antinodes):** அதிகபட்ச வீச்சு கொண்ட புள்ளிகள்.
    *   திறந்த ஆர்கன் குழாய் (Open Organ Pipe), மூடிய ஆர்கன் குழாய் (Closed Organ Pipe), ஒரு முனையில் கட்டப்பட்ட கம்பி (String fixed at one end).

    ```
      Λ    Λ    Λ    Λ
     / \  / \  / \  / \
    N   AN   N   AN   N   AN   N   AN   N
    \ /  \ /  \ /  \ /
     V    V    V    V
    ```
    *நிலை அலை - Standing Wave (கணு (N), எதிர்கணு (AN))*
    **தமிழ்த் தொடர்பு:** வீடுகளில் பயன்படுத்தப்படும் சரம் கட்டப்பட்ட கருவிகளில் (like Veena or Guitar) உருவாகும் இசை அலைகள்.

#### டாப்ளர் விளைவு (Doppler Effect):

ஒலி மூலத்திற்கும் (source) கேட்பவருக்கும் (observer) இடையே சார்பு இயக்கம் (relative motion) இருக்கும்போது, கேட்கும் அதிர்வெண்ணில் (apparent frequency) ஒரு மாற்றம் ஏற்படுவதே டாப்ளர் விளைவு எனப்படும்.
$f' = f \left( \frac{v \pm v_o}{v \mp v_s} \right)$
இங்கு, $f'$ என்பது கேட்கும் அதிர்வெண், $f$ என்பது உண்மையான அதிர்வெண், $v$ என்பது ஒலியின் வேகம், $v_o$ என்பது கேட்பவரின் வேகம், $v_s$ என்பது மூலத்தின் வேகம்.
(மேல் குறியீடு 'மூலம் கேட்குபவரை நோக்கி வரும்போது' அல்லது 'கேட்குபவர் மூலத்தை நோக்கி செல்லும்போது' பயன்படுத்தவும், கீழ் குறியீடு 'மூலம் கேட்குபவரை விட்டு விலகும்போது' அல்லது 'கேட்குபவர் மூலத்தை விட்டு விலகும்போது' பயன்படுத்தவும்).

**தமிழ்த் தொடர்பு:** ரயில் நிலையம் அல்லது பேருந்து நிலையத்தில் ரயில் அல்லது பேருந்து அருகில் வரும்போது ஒலி சத்தமாகவும், அதிர்வெண் அதிகமாகவும் இருக்கும்; விலகிச் செல்லும்போது மெதுவாகவும், அதிர்வெண் குறைவாகவும் இருக்கும். (தனியார் பேருந்து ஹார்ன் ஒலி).

## Worked Examples (தீர்க்கப்பட்ட எடுத்துக்காட்டுகள்)

**எடுத்துக்காட்டு 1:**
ஒரு பொருள் எளிய சீரிசை இயக்கத்தில் அலைவுறுகிறது. அதன் வீச்சு 10 cm மற்றும் அலைவுக்காலம் 2 s. $t=0$ இல் பொருள் அதன் சமநிலை புள்ளியில் இருந்தால், $t=0.5$ s இல் அதன் இடப்பெயர்ச்சி என்ன?

**தீர்வு:**
கொடுக்கப்பட்டவை: $A = 10 \text{ cm}$, $T = 2 \text{ s}$.
$t=0$ இல் $x=0$ என்பதால், எளிய சீரிசை இயக்கத்தின் சமன்பாடு $x(t) = A \sin(\omega t)$.
கோண அதிர்வெண் $ω = \frac{2\pi}{T} = \frac{2\pi}{2} = \pi \text{ rad/s}$.

இடப்பெயர்ச்சி சமன்பாடு: $x(t) = 10 \sin(\pi t)$.
$t=0.5$ s இல்,
$x(0.5) = 10 \sin(\pi \times 0.5) = 10 \sin(\frac{\pi}{2}) = 10 \times 1 = 10 \text{ cm}$.

**எடுத்துக்காட்டு 2:**
ஒரு 2 kg நிறை கொண்ட பொருள் 200 N/m ஸ்பிரிங் மாறிலி கொண்ட ஸ்பிரிங்கில் இணைக்கப்பட்டு, கிடைமட்ட மேற்பரப்பில் எளிய சீரிசை இயக்கத்தில் அலைவுறுகிறது. அதன் அதிகபட்ச திசைவேகம் 10 m/s எனில், அதன் வீச்சு மற்றும் மொத்த ஆற்றலைக் கணக்கிடுக.

**தீர்வு:**
கொடுக்கப்பட்டவை: $m = 2 \text{ kg}$, $k = 200 \text{ N/m}$, $v_{max} = 10 \text{ m/s}$.
அதிகபட்ச திசைவேகம் $v_{max} = A\omega$.
$ω = \sqrt{\frac{k}{m}} = \sqrt{\frac{200}{2}} = \sqrt{100} = 10 \text{ rad/s}$.

வீச்சு $A = \frac{v_{max}}{\omega} = \frac{10}{10} = 1 \text{ m}$.

மொத்த ஆற்றல் $E = \frac{1}{2} k A^2 = \frac{1}{2} (200) (1)^2 = 100 \text{ J}$.

**எடுத்துக்காட்டு 3:**
ஒரு எளிய ஊசலின் நீளம் 1 m. புவி ஈர்ப்பு முடுக்கம் $g = 9.8 \text{ m/s}^2$ எனில், அதன் அலைவுக்காலம் என்ன?

**தீர்வு:**
கொடுக்கப்பட்டவை: $L = 1 \text{ m}$, $g = 9.8 \text{ m/s}^2$.
எளிய ஊசலின் அலைவுக்காலம் $T = 2\pi \sqrt{\frac{L}{g}}$.
$T = 2\pi \sqrt{\frac{1}{9.8}} \approx 2\pi \sqrt{0.102} \approx 2 \times 3.1415 \times 0.319 \approx 2.00 \text{ s}$.

**எடுத்துக்காட்டு 4:**
ஒரு கயிற்றில் 10 m/s வேகத்தில் செல்லும் ஒரு குறுக்கலையின் அதிர்வெண் 5 Hz எனில், அதன் அலைநீளம் என்ன?

**தீர்வு:**
கொடுக்கப்பட்டவை: $v = 10 \text{ m/s}$, $f = 5 \text{ Hz}$.
அலை வேகம் $v = f\lambda$.
அலைநீளம் $\lambda = \frac{v}{f} = \frac{10}{5} = 2 \text{ m}$.

**எடுத்துக்காட்டு 5:**
ஒரு கம்பி 20 N இழுவிசையுடன் உள்ளது மற்றும் அதன் ஓரலகு நீளத்திற்கான நிறை $0.05 \text{ kg/m}$. கம்பியில் குறுக்கலையின் வேகம் என்ன?

**தீர்வு:**
கொடுக்கப்பட்டவை: $Tension (\text{இழுவிசை}) = 20 \text{ N}$, $\mu = 0.05 \text{ kg/m}$.
குறுக்கலை வேகம் $v = \sqrt{\frac{\text{இழுவிசை}}{\mu}} = \sqrt{\frac{20}{0.05}} = \sqrt{\frac{20}{1/20}} = \sqrt{20 \times 20} = \sqrt{400} = 20 \text{ m/s}$.

**எடுத்துக்காட்டு 6:**
இரண்டு ஒலி மூலங்கள் 320 Hz மற்றும் 324 Hz அதிர்வெண்களை வெளியிடுகின்றன. இவற்றின் மேற்பொருந்துதலால் ஏற்படும் விம்மல் அதிர்வெண் என்ன?

**தீர்வு:**
கொடுக்கப்பட்டவை: $f_1 = 320 \text{ Hz}$, $f_2 = 324 \text{ Hz}$.
விம்மல் அதிர்வெண் $f_{beat} = |f_1 - f_2| = |320 - 324| = |-4| = 4 \text{ Hz}$.

**எடுத்துக்காட்டு 7:**
ஒரு ரயில் 330 m/s வேகத்தில் ஒலிபரப்பிக் கொண்டு, 10 m/s வேகத்தில் ஒரு நிலையான கேட்பவரை நோக்கி வருகிறது. ரயிலின் உண்மையான அதிர்வெண் 1000 Hz எனில், கேட்பவர் கேட்கும் அதிர்வெண் என்ன? (ஒலியின் வேகம் = 330 m/s).

**தீர்வு:**
கொடுக்கப்பட்டவை: $v = 330 \text{ m/s}$, $v_s = 10 \text{ m/s}$ (மூலம் கேட்பவரை நோக்கி வருகிறது), $v_o = 0$ (கேட்பவர் நிலையாக உள்ளார்), $f = 1000 \text{ Hz}$.
டாப்ளர் விளைவுக்கான சமன்பாடு: $f' = f \left( \frac{v \pm v_o}{v \mp v_s} \right)$.
மூலம் கேட்பவரை நோக்கி வருவதால், знаменаட்டியில் $v-v_s$ பயன்படுத்த வேண்டும். கேட்பவர் நிலையாக இருப்பதால் $v_o=0$.
$f' = f \left( \frac{v}{v - v_s} \right) = 1000 \left( \frac{330}{330 - 10} \right) = 1000 \left( \frac{330}{320} \right) = 1000 \times \frac{33}{32} \approx 1031.25 \text{ Hz}$.

### MCQs Solutions (பலவுள் தெரிவு வினாக்களின் விடைகள்)

**1.** b) இடப்பெயர்ச்சி
    *   **விளக்கம்:** $a = -\omega^2 x$. எனவே, முடுக்கம் இடப்பெயர்ச்சிக்கு நேர் விகிதத்தில் இருக்கும்.

**2.** b) வீச்சின் இருமடி
    *   **விளக்கம்:** $E = \frac{1}{2} k A^2$.

**3.** b) அதன் நீளத்தின் இருமடி மூலம்
    *   **விளக்கம்:** $T = 2\pi \sqrt{\frac{L}{g}}$.

**4.** c) $T = 2\pi \sqrt{\frac{m}{k}}$
    *   **விளக்கம்:** ஸ்பிரிங்-நிறை அமைப்பின் அலைவுக்காலத்திற்கான சூத்திரம்.

**5.** c) தணிந்த அலைவுகள்
    *   **விளக்கம்:** ஆற்றல் இழப்பால் வீச்சு குறைவது தணிந்த அலைவுகள்.

**6.** c) ஆற்றல்
    *   **விளக்கம்:** அலைகள் ஆற்றலை மட்டுமே கடத்துகின்றன, ஊடக துகள்களை அல்ல.

**7.** b) குறுக்கலைகள்
    *   **விளக்கம்:** குறுக்கலைகளில் துகள்கள் பரவல் திசைக்கு செங்குத்தாக அலைவுறும்.

**8.** c) $v = f \lambda$
    *   **விளக்கம்:** அலை வேகம், அதிர்வெண் மற்றும் அலைநீளத்திற்கு இடையேயான அடிப்படை உறவு.

**9.** a) அதிகரிக்கும்

**10.** a) அதிர்வெண் மாற்றம்
    *   **விளக்கம்:** டாப்ளர் விளைவு, சார்பு இயக்கத்தால் கேட்கும் அதிர்வெண்ணில் ஏற்படும் மாற்றத்தை விவரிக்கிறது.

**11.** b) கயிற்றின் இழுவிசை மற்றும் ஓரலகு நீளத்திற்கான நிறை
    *   **விளக்கம்:** $v = \sqrt{\frac{T}{\mu}}$.

**12.** b) எதிர்கணுக்கள் (Antinodes)
    *   **விளக்கம்:** எதிர்கணுக்கள் அதிகபட்ச வீச்சு கொண்ட புள்ளிகள்.

    *   **விளக்கம்:** டாப்ளர் விளைவின் வரையறை.

**14.** b) ஆக்கபூர்வ குறுக்கீடு
    *   **விளக்கம்:** ஒரே கட்டத்தில் அலைகள் கூடும்போது ஆக்கபூர்வ குறுக்கீடு ஏற்படுகிறது.

    *   **விளக்கம்:** $f' = f \left( \frac{v}{v - v_s} \right) = 500 \left( \frac{340}{340 - 20} \right) = 500 \left( \frac{340}{320} \right) = 500 \times \frac{17}{16} = 531.25 \text{ Hz}$.

**16.** b) வீச்சு நிலைகளில் (endpoints)
    *   **விளக்கம்:** $a = -\omega^2 x$. $x$ அதிகபட்சமாக (A) இருக்கும்போது, $a$ அதிகபட்சமாக இருக்கும்.

**17.** a) சமநிலை புள்ளியில்
    *   **விளக்கம்:** $v = \pm \omega \sqrt{A^2 - x^2}$. $x=0$ இல் $v = \pm A\omega$, இது அதிகபட்சம்.

**18.** b) 4
    *   **விளக்கம்:** $f_{beat} = |f_1 - f_2| = |256 - 260| = 4 \text{ Hz}$.

**19.** b) குறுக்கலைகள்

**20.** c) அதிர்வெண்

**21.** c) $3f$
    *   **விளக்கம்:** மூடிய ஆர்கன் குழாயில் ஒற்றைப்படை மேலதிர்வுகள் மட்டுமே சாத்தியம்: $f, 3f, 5f, ...$.

**22.** c) அலை வேகம்
    *   **விளக்கம்:** $v = \frac{\omega}{k}$.

**23.** a) 5 rad/s
    *   **விளக்கம்:** $v_{max} = A\omega \implies \omega = \frac{v_{max}}{A} = \frac{50 \text{ cm/s}}{10 \text{ cm}} = 5 \text{ rad/s}$.

**24.** b) இரட்டிப்பாகும்
    *   **விளக்கம்:** $T = 2\pi \sqrt{\frac{L}{g}}$. $L$ ஐ $4L$ ஆக்கினால், $T' = 2\pi \sqrt{\frac{4L}{g}} = 2 \times (2\pi \sqrt{\frac{L}{g}}) = 2T$.

**25.** c) இயற்கையான அதிர்வெண்
    *   **விளக்கம்:** ஒத்ததிர்வின் வரையறை.

## Assertion–Reason Questions (உறுதிப்படுத்தல் - காரணம் வினாக்கள்)

c) A உண்மை, ஆனால் R தவறு.
d) A தவறு, ஆனால் R உண்மை.

**R:** சமநிலை புள்ளியில் இயக்க ஆற்றல் அதிகபட்சமாக இருக்கும்.
    a) A மற்றும் R இரண்டும் உண்மை, மேலும் R என்பது A வின் சரியான விளக்கம்.
    b) A மற்றும் R இரண்டும் உண்மை, ஆனால் R என்பது A வின் சரியான விளக்கம் அல்ல.

### Assertion–Reason Solutions (உறுதிப்படுத்தல் - காரணம் வினாக்களின் விடைகள்)

**1.** a) A மற்றும் R இரண்டும் உண்மை, மேலும் R என்பது A வின் சரியான விளக்கம்.
    *   **விளக்கம்:** ஸ்பிரிங்-நிறை அமைப்பின் அலைவுக்கால சூத்திரத்தில் $g$ இல்லை. எனவே, A உண்மை. R என்பது சூத்திரம், இது A வுக்கு சரியான விளக்கம்.

**2.** a) A மற்றும் R இரண்டும் உண்மை, மேலும் R என்பது A வின் சரியான விளக்கம்.
    *   **விளக்கம்:** சமநிலை புள்ளியில் ($x=0$), $PE=0$ மற்றும் $KE$ அதிகபட்சம். $KE$ அதிகபட்சமாக இருக்கும்போது, திசைவேகம் அதிகபட்சமாக இருக்கும். எனவே, A மற்றும் R இரண்டும் உண்மை மற்றும் R என்பது A வின் சரியான விளக்கம்.

**3.** a) A மற்றும் R இரண்டும் உண்மை, மேலும் R என்பது A வின் சரியான விளக்கம்.
    *   **விளக்கம்:** அலைவுக்காலம் $T = 2\pi \sqrt{L/g}$. $g=0$ (பூஜ்ஜிய ஈர்ப்பு) என்றால் $T$ முடிவிலியாக இருக்கும். அலைவு இயக்கம் நிகழ ஈர்ப்பு விசை அவசியம். எனவே A மற்றும் R இரண்டும் உண்மை மற்றும் R என்பது A வின் சரியான விளக்கம்.

**4.** a) A மற்றும் R இரண்டும் உண்மை, மேலும் R என்பது A வின் சரியான விளக்கம்.
    *   **விளக்கம்:** ஒத்ததிர்வின் வரையறை. ஒத்ததிர்வில், இயக்கும் அதிர்வெண் அமைப்பின் இயற்கையான அதிர்வெண்ணுடன் பொருந்தும்போது ஆற்றல் பரிமாற்றம் அதிகமாகி, வீச்சு கூடும்.

**5.** c) A உண்மை, ஆனால் R தவறு.
    *   **விளக்கம்:** அலைகள் ஆற்றலை மாற்றுகின்றன, ஆனால் ஊடகத்தின் துகள்களை நிரந்தரமாக இடமாற்றம் செய்யாது. துகள்கள் தங்கள் சமநிலை புள்ளிகளைச் சுற்றி அலைவுறும்.

**6.** a) A மற்றும் R இரண்டும் உண்மை, மேலும் R என்பது A வின் சரியான விளக்கம்.

    *   **விளக்கம்:** கணுக்களில், துகள்கள் நகரவில்லை (இடப்பெயர்ச்சி பூஜ்ஜியம்). ஆற்றல் பரிமாற்றம் துகள்களின் இயக்கத்துடன் தொடர்புடையது, எனவே கணுக்களில் ஆற்றல் கடத்தப்படுவதில்லை.

**9.** b) A மற்றும் R இரண்டும் உண்மை, ஆனால் R என்பது A வின் சரியான விளக்கம் அல்ல.
    *   **விளக்கம்:** $v = f\lambda$. ஒரு குறிப்பிட்ட அலை வேகத்திற்கு, $\lambda = v/f$, எனவே அலைநீளம் அதிர்வெண்ணுக்கு நேர்மாறாக இருக்கும் (A உண்மை). R என்பது அலை வேகம், அலைநீளம் மற்றும் அதிர்வெண் இடையே உள்ள உறவை சரியாகக் கூறுகிறது (R உண்மை). ஆனால் R என்பது A வின் நேரடி விளக்கம் அல்ல; அது ஒரு பொதுவான சூத்திரம்.

**10.** d) A தவறு, ஆனால் R உண்மை.
    *   **விளக்கம்:** எதிரெதிர் கட்டத்தில் மேற்பொருந்தும்போது அழிப்பபூர்வ குறுக்கீடு ஏற்படும். ஆக்கபூர்வ குறுக்கீடு என்பது அலைகள் ஒன்றுக்கொன்று வலுவூட்டுவது. எனவே A தவறு, R உண்மை.

**11.** a) A மற்றும் R இரண்டும் உண்மை, மேலும் R என்பது A வின் சரியான விளக்கம்.
    *   **விளக்கம்:** அலை ஒரு ஊடகத்திலிருந்து மற்றொரு ஊடகத்திற்கு செல்லும் போது அதன் அதிர்வெண் மாறாமல் இருக்கும். இது மூலத்தின் அதிர்வெண்ணைப் பொறுத்தது, ஊடகத்தைப் பொறுத்தது அல்ல.

**12.** c) A உண்மை, ஆனால் R தவறு.
    *   **விளக்கம்:** ஒலி அலைகள் இயந்திர அலைகள், அவை பயணிக்க ஒரு ஊடகம் தேவை. அவை வெற்றிடத்தின் வழியாக பயணிக்க முடியாது.

**13.** a) A மற்றும் R இரண்டும் உண்மை, மேலும் R என்பது A வின் சரியான விளக்கம்.
    *   **விளக்கம்:** மூடிய ஆர்கன் குழாயின் ஒரு முனை மூடப்பட்டு, மற்றொரு முனை திறந்திருக்கும். மூடிய முனையில் எப்போதும் ஒரு கணுவும், திறந்த முனையில் ஒரு எதிர்கணுவும் உருவாகும். இந்த நிபந்தனை ஒற்றைப்படை மேலதிர்வுகளுக்கு மட்டுமே வழிவகுக்கிறது.

**14.** c) A உண்மை, ஆனால் R தவறு.
    *   **விளக்கம்:** $v=0$ (வீச்சு நிலைகளில்) இருக்கும்போது, $x$ அதிகபட்சம் ($A$), இதனால் $a = -\omega^2 x$ அதிகபட்சம். எனவே A உண்மை. R இல், திசைவேகம் இடப்பெயர்ச்சி பூஜ்ஜியமாக இருக்கும்போது அதிகபட்சமாக இருக்கும் என்று கூறப்படுகிறது, இது உண்மை. ஆனால் 'மற்றும் திசைவேகம் இடப்பெயர்ச்சி பூஜ்ஜியமாக இருக்கும்போது அதிகபட்சமாக இருக்கும்' என்பது A வுக்கான காரணம் அல்ல, மாறாக 'முடுக்கம் இடப்பெயர்ச்சிக்கு நேர் விகிதத்தில் இருக்கும்' என்பதோடு தொடர்புடையது. (திசைவேகம் பூஜ்ஜியமாக இருக்கும்போது $x$ அதிகபட்சம், முடுக்கம் அதிகபட்சம்; திசைவேகம் அதிகபட்சமாக இருக்கும்போது $x$ பூஜ்ஜியம், முடுக்கம் பூஜ்ஜியம்).

**15.** a) A மற்றும் R இரண்டும் உண்மை, மேலும் R என்பது A வின் சரியான விளக்கம்.
    *   **விளக்கம்:** விம்மல் அதிர்வெண் $f_{beat} = |f_1 - f_2|$. $f_1 = f_2$ என்றால் $f_{beat}=0$, அதாவது விம்மல்கள் இல்லை. எனவே, வேறுபட்ட அதிர்வெண்கள் தேவை.
