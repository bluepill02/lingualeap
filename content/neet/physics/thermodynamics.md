# NEET Physics: Thermodynamics - வெப்ப இயக்கவியல்

## 1. Concept Notes (கருத்து குறிப்புகள்)

### 1.1 Introduction to Thermodynamics (வெப்ப இயக்கவியலின் அறிமுகம்)

வெப்ப இயக்கவியல் என்பது ஆற்றல் மற்றும் அதன் மாற்றங்கள், குறிப்பாக வெப்பம் (Heat) மற்றும் வேலை (Work) ஆகியவற்றுக்கு இடையேயான தொடர்புகளைப் பற்றிப் படிக்கும் இயற்பியலின் ஒரு கிளை ஆகும்.

*   **அமைப்பு (System):** நாம் ஆய்வு செய்யத் தேர்ந்தெடுக்கும் பிரபஞ்சத்தின் ஒரு பகுதி. எ.கா: ஒரு பாத்திரத்தில் உள்ள நீர்.
*   **சுற்றுப்புறம் (Surroundings):** அமைப்பைத் தவிர மற்ற பிரபஞ்சத்தின் பகுதி.
*   **எல்லை (Boundary):** அமைப்பையும் சுற்றுப்புறத்தையும் பிரிக்கும் ஒரு உண்மையான அல்லது கற்பனையான மேற்பரப்பு.

**அமைப்புகளின் வகைகள் (Types of Systems):**
1.  **திறந்த அமைப்பு (Open System):** ஆற்றலையும் (Energy), பொருளையும் (Matter) சுற்றுப்புறத்துடன் பரிமாறிக்கொள்ளும். எ.கா: திறந்த பாத்திரத்தில் கொதிக்கும் நீர்.
2.  **மூடிய அமைப்பு (Closed System):** ஆற்றலை மட்டும் பரிமாறிக்கொள்ளும், ஆனால் பொருளைப் பரிமாறிக்கொள்ளாது. எ.கா: மூடப்பட்ட பாத்திரத்தில் சூடான நீர்.
3.  **தனிமைப்படுத்தப்பட்ட அமைப்பு (Isolated System):** ஆற்றலையோ, பொருளையோ பரிமாறிக்கொள்ளாது. எ.கா: ஒரு சிறந்த வெப்பக்கடத்தாத கொள்கலனில் (Thermos flask) உள்ள நீர்.

**வெப்ப இயக்கவியல் மாறிகள் (Thermodynamic Variables):** ஒரு அமைப்பின் நிலையை விவரிக்கப் பயன்படும் அளவுகள். முக்கிய மாறிகள்: அழுத்தம் (Pressure - P), கன அளவு (Volume - V), வெப்பநிலை (Temperature - T), மோல்களின் எண்ணிக்கை (Number of moles - n).

**வெப்ப இயக்கவியல் சமநிலை (Thermodynamic Equilibrium):** ஒரு அமைப்பு இயந்திரவியல் சமநிலை (mechanical equilibrium), ரசாயன சமநிலை (chemical equilibrium) மற்றும் வெப்ப சமநிலை (thermal equilibrium) ஆகிய மூன்றிலும் இருக்கும்போது அது வெப்ப இயக்கவியல் சமநிலையில் உள்ளது என்று கூறப்படுகிறது.

### 1.2 Zeroth Law of Thermodynamics (வெப்ப இயக்கவியலின் பூஜ்ஜிய விதி)

**விதி:** இரண்டு அமைப்புகள் தனித்தனியே ஒரு மூன்றாவது அமைப்புடன் வெப்ப சமநிலையில் இருந்தால், அவை ஒன்றுடன் ஒன்று வெப்ப சமநிலையில் இருக்கும்.

**கருத்து:** இது வெப்பநிலையின் கருத்தை வரையறுக்கிறது. ஒரே வெப்பநிலையில் உள்ள பொருட்கள் வெப்பப் பரிமாற்றம் இன்றி சமநிலையில் இருக்கும்.

**தமிழ் உதாரணம் (Tamil Analogy):** உங்கள் நண்பருக்கும் உங்களுக்கும் காய்ச்சல் இருப்பதாக வைத்துக்கொள்வோம். டாக்டர் உங்கள் இருவருக்கும் ஒரே வெப்பமானியைப் பயன்படுத்தி வெப்பநிலையை அளக்கிறார். இருவருக்கும் ஒரே வெப்பநிலை காட்டப்பட்டால், நீங்கள் இருவரும் வெப்ப சமநிலையில் இருக்கிறீர்கள் என்று அர்த்தம். இங்கு வெப்பமானி மூன்றாவது அமைப்பாக செயல்படுகிறது.

### 1.3 First Law of Thermodynamics (வெப்ப இயக்கவியலின் முதல் விதி)

**விதி:** ஆற்றல் அழிவின்மை விதிக்கு சமம். ஒரு அமைப்பிற்கு அளிக்கப்படும் வெப்ப ஆற்றல் (Q) அதன் அக ஆற்றலை அதிகரிக்கவும் ($\Delta U$), சுற்றுப்புறத்தின் மீது வேலை செய்யவும் (W) பயன்படுகிறது.

**சமன்பாடு:** $\Delta U = Q - W$

*   **$\Delta U$ (அக ஆற்றல் மாற்றம்):** அமைப்பின் மூலக்கூறுகளின் இயக்க மற்றும் நிலை ஆற்றல்களின் கூட்டுத்தொகை. இது அமைப்பின் வெப்பநிலை சார்ந்திருக்கும்.
*   **Q (அமைப்புக்கு அளிக்கப்படும் வெப்பம்):**
    *   $Q > 0$: அமைப்பு வெப்பத்தை உறிஞ்சினால் (உள்ளே).
    *   $Q < 0$: அமைப்பு வெப்பத்தை வெளியிட்டால் (வெளியே).
*   **W (அமைப்பினால் செய்யப்பட்ட வேலை):**
    *   $W > 0$: அமைப்பு வேலை செய்தால் (விரிவாக்கம்).
    *   $W < 0$: அமைப்பின் மீது வேலை செய்யப்பட்டால் (சுருக்கம்).

**வெப்ப இயக்கவியல் செயல்முறைகள் (Thermodynamic Processes):**

1.  **சமவெப்ப செயல்முறை (Isothermal Process):** வெப்பநிலை மாறாமல் இருக்கும் ($\Delta T = 0$). எனவே, அக ஆற்றல் மாற்றம் $\Delta U = 0$. முதல் விதியின்படி, $Q = W$.
    *   `P-V diagram for Isothermal process (சமவெப்ப செயல்முறைக்கு P-V வரைபடம்):`
    ```
         P |
           |    _ _ _ _ _ _ _ _ _ _ _ _ _
           |   /                           \ 
           |  /                             \  T = மாறிலி (Constant)
           | /                               \ 
           +---------------------------------- V
    ```

2.  **அடியபேடிக் செயல்முறை (Adiabatic Process):** அமைப்பிற்கும் சுற்றுப்புறத்திற்கும் இடையே வெப்பப் பரிமாற்றம் இல்லை ($Q = 0$). முதல் விதியின்படி, $\Delta U = -W$.
    *   `P-V diagram for Adiabatic process (அடியபேடிக் செயல்முறைக்கு P-V வரைபடம்):`
    ```
         P |
           |    _ _ _ _ _ _ _ _ _
           |   /                   \ 
           |  /                     \  Q = 0
           | /                       \ 
           +-------------------------- V
    ```

3.  **சமஅழுத்த செயல்முறை (Isobaric Process):** அழுத்தம் மாறாமல் இருக்கும் ($\Delta P = 0$). அமைப்பினால் செய்யப்படும் வேலை $W = P\Delta V$.
    *   `P-V diagram for Isobaric process (சமஅழுத்த செயல்முறைக்கு P-V வரைபடம்):`
    ```
         P |   -------------------
           |  |                   |  P = மாறிலி (Constant)
           |  |                   |
           |  |                   |
           +-------------------------- V
    ```

4.  **சமகனஅளவு செயல்முறை (Isochoric Process):** கன அளவு மாறாமல் இருக்கும் ($\Delta V = 0$). எனவே, செய்யப்படும் வேலை $W = 0$. முதல் விதியின்படி, $\Delta U = Q$.
    *   `P-V diagram for Isochoric process (சமகனஅளவு செயல்முறைக்கு P-V வரைபடம்):`
    ```
         P |   ^
           |   |
           |   | V = மாறிலி (Constant)
           |   |
           +-------------------------- V
    ```

5.  **சுழற்சி செயல்முறை (Cyclic Process):** ஒரு அமைப்பானது தொடக்க நிலைக்குத் திரும்பி வரும் செயல்முறை. இதில் அக ஆற்றல் மாற்றம் $\Delta U = 0$. எனவே, $Q = W$.

**தமிழ் உதாரணம் (Tamil Analogy):** பிரஷர் குக்கர் (Pressure Cooker). பிரஷர் குக்கரில் உள்ள உணவு சூடாகி, நீராவி உருவாகிறது. இந்த நீராவி குக்கரின் உள்ளே அழுத்தம் கொடுத்து மூடியைத் தூக்க முயல்கிறது (வேலை செய்யப்படுகிறது). அதே நேரத்தில், குக்கரின் அடிப்பகுதியில் இருந்து வெப்பம் உள்ளே செல்கிறது. குக்கரில் உள்ள உணவின் அக ஆற்றல் (temperature of food) அதிகரிக்கிறது. இது வெப்பம் (Q) மற்றும் வேலை (W) மூலம் உள்வரும் மற்றும் வெளியேறும் ஆற்றலை விளக்குகிறது.

### 1.4 Second Law of Thermodynamics (வெப்ப இயக்கவியலின் இரண்டாம் விதி)

இந்த விதி வெப்ப ஓட்டத்தின் திசை மற்றும் ஒரு செயல்முறையின் இயல்பு தன்மையை விளக்குகிறது. முழுமையான ஆற்றல் மாற்றத்தின் சாத்தியமற்ற தன்மையை இது கோடிட்டுக் காட்டுகிறது.

**1. கெல்வின்-பிளாங்க் கூற்று (Kelvin-Planck Statement):** வெப்ப மூலத்திலிருந்து வெப்பத்தை உறிஞ்சி, அதை முழுவதுமாக வேலையாக மாற்றும் ஒரு இயந்திரத்தை (100% செயல்திறன் கொண்ட வெப்ப இயந்திரம்) உருவாக்குவது சாத்தியமற்றது.

**2. கிளாசியஸ் கூற்று (Clausius Statement):** எந்தவொரு வெளிவிசையின் உதவியும் இன்றி, வெப்பம் தானாகவே ஒரு குளிர்ந்த பொருளிலிருந்து சூடான பொருளுக்குப் பாயாது.

**என்ட்ரோபி (Entropy):** ஒரு அமைப்பின் ஒழுங்கின்மையின் (அல்லது குழப்பத்தின்) அளவு. பிரபஞ்சத்தின் என்ட்ரோபி எப்போதும் அதிகரிக்கிறது.

**வெப்ப இயந்திரம் (Heat Engine):** உயர் வெப்பநிலையிலிருந்து ($T_1$) வெப்பத்தை ($Q_1$) உறிஞ்சி, அதன் ஒரு பகுதியை வேலையாக (W) மாற்றி, மீதமுள்ள வெப்பத்தை ($Q_2$) குறைந்த வெப்பநிலைக்கு ($T_2$) வெளியிடும் கருவி.

*   `Heat Engine Diagram (வெப்ப இயந்திர வரைபடம்):`
    ```
        உயர் வெப்பநிலை மூலம் (High Temp Source) T1
                       | 
                       | Q1 (உறிஞ்சப்பட்ட வெப்பம் - Heat Absorbed)
                       V
                  [வெப்ப இயந்திரம் - Heat Engine]
                 /    \ 
                /      \ 
               W (செய்யப்பட்ட வேலை - Work Done)
              /        \ 
             V          V
             Q2 (வெளியிடப்பட்ட வெப்பம் - Heat Released)
                       | 
                       | 
        குறைந்த வெப்பநிலை ஒடுக்கி (Low Temp Sink) T2
    ```
*   **செயல்திறன் (Efficiency), $\eta$:** $\eta = \frac{\text{செய்யப்பட்ட வேலை}}{\text{உறிஞ்சப்பட்ட வெப்பம்}} = \frac{W}{Q_1} = \frac{Q_1 - Q_2}{Q_1} = 1 - \frac{Q_2}{Q_1}$
    *   கார்னோ இயந்திரத்திற்கு, $\eta = 1 - \frac{T_2}{T_1}$ (இங்கு $T_1$ மற்றும் $T_2$ கெல்வின் அலகில் இருக்க வேண்டும்).

**குளிர்சாதனப் பெட்டி (Refrigerator) / வெப்பப் பம்ப் (Heat Pump):** ஒரு அமைப்பிலிருந்து வெப்பத்தை அகற்றி, அதை உயர் வெப்பநிலைக்கு மாற்ற வெளி வேலை தேவைப்படுகிறது (கிளாசியஸ் கூற்றின்படி).

*   **செயல்திறன் குணகம் (Coefficient of Performance, COP), $\beta$:** $\beta = \frac{\text{குளிர்விக்கப்பட்ட பகுதியிலிருந்து நீக்கப்பட்ட வெப்பம்}}{\text{செய்யப்பட்ட வேலை}} = \frac{Q_2}{W} = \frac{Q_2}{Q_1 - Q_2}$
    *   கார்னோ குளிர்சாதனப் பெட்டிக்கு, $\beta = \frac{T_2}{T_1 - T_2}$.

**தமிழ் உதாரணம் (Tamil Analogy):** கீழே பாயும் நீர் (Water flowing downhill). நீர் எப்போதும் தானாகவே உயரத்திலிருந்து பள்ளத்தை நோக்கிப் பாயும். பள்ளத்திலிருந்து உயரத்திற்கு நீர் பாய வேண்டுமானால், நாம் மோட்டார் போன்ற ஒரு கருவியைப் பயன்படுத்தி வேலை செய்ய வேண்டும். இது இரண்டாம் விதியின் என்ட்ரோபி கருத்தை ஒத்திருக்கிறது – ஆற்றல் எப்போதும் அதிக ஒழுங்கின்மை நிலையை (குழப்பத்தை) நோக்கி நகர்கிறது, அல்லது பயனுள்ள வேலைக்கு ஆற்றல் மாற்றம் ஒரு குறிப்பிட்ட திசையில் மட்டுமே நடக்கும்.

### 1.5 Third Law of Thermodynamics (வெப்ப இயக்கவியலின் மூன்றாம் விதி)

**விதி:** எந்தவொரு செயல்முறை மூலமும் ஒரு அமைப்பை முழுமையான பூஜ்ஜிய வெப்பநிலைக்கு (0 கெல்வின் அல்லது $-273.15 \text{°C}$) கொண்டு செல்ல முடியாது. முழுமையான பூஜ்ஜிய வெப்பநிலையில், ஒரு சரியான படிகத்தின் என்ட்ரோபி பூஜ்ஜியமாக இருக்கும்.

**கருத்து:** இது முழுமையான பூஜ்ஜியத்தின் அணுக முடியாத தன்மையைக் குறிக்கிறது.
