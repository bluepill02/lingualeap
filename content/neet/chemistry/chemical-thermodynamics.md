# NEET வேதியியல்: வெப்ப இயக்கவியல் (Chemical Thermodynamics)

## தொகுதி 1: கருத்து குறிப்புகள் (Concept Notes)

வெப்ப இயக்கவியல் என்பது இயற்பியல் வேதியியலின் ஒரு கிளை ஆகும், இது ஆற்றல் மாற்றங்கள் மற்றும் அவை ஒரு அமைப்பின் பண்புகளை எவ்வாறு பாதிக்கின்றன என்பதைப் படிக்கிறது. வேதிவினைகள் ஏன் நிகழ்கின்றன (தன்னிச்சையானது - Spontaneous) அல்லது நிகழாது என்பதைப் புரிந்துகொள்ள இது உதவுகிறது.

### 1. அடிப்படை வரையறைகள் (Basic Definitions)

*   **அமைப்பு (System):** நாம் ஆய்வு செய்யும் பிரபஞ்சத்தின் ஒரு குறிப்பிட்ட பகுதி. (தமிழில்: ஒரு தேநீர் கோப்பை, ஒரு ரசாயன உலை)
*   **சுற்றுப்புறம் (Surroundings):** அமைப்புக்கு வெளியே உள்ள பிரபஞ்சத்தின் மற்ற அனைத்து பகுதிகளும். (தமிழில்: கோப்பையைச் சுற்றியுள்ள அறை, உலையைச் சுற்றியுள்ள தொழிற்சாலை)
*   **எல்லை (Boundary):** அமைப்பையும் சுற்றுப்புறத்தையும் பிரிக்கும் ஒரு உண்மையான அல்லது கற்பனையான கோடு. (தமிழில்: தேநீர் கோப்பையின் சுவர், உலைக்கான உறை)

```
       +--------------------+
       |   சுற்றுப்புறம்    |
       |  (Surroundings)    |
       | +----------------+ |
       | |     அமைப்பு     | |
       | |     (System)     | |
       | +----------------+ |
       +--------------------+
         ^
         |
         எல்லை (Boundary)
```

### 2. அமைப்புகளின் வகைகள் (Types of Systems)

*   **திறந்த அமைப்பு (Open System):** அமைப்பு சுற்றுப்புறங்களுடன் ஆற்றல் (வெப்பம்) மற்றும் பொருள் (பொருளின் நிறை) இரண்டையும் பரிமாறிக்கொள்ள முடியும்.
    *   **உதாரணம்:** சூடான தேநீர் நிரப்பப்பட்ட திறந்த கோப்பை. (வெப்பம் வெளியாகும், நீராவி வெளியேறும் - பொருள் வெளியேறும்)
*   **மூடிய அமைப்பு (Closed System):** அமைப்பு சுற்றுப்புறங்களுடன் ஆற்றலை மட்டுமே பரிமாறிக்கொள்ள முடியும், ஆனால் பொருளை பரிமாறிக்கொள்ள முடியாது.
    *   **உதாரணம்:** சூடான தேநீர் நிரப்பப்பட்ட மூடிய கோப்பை. (வெப்பம் வெளியாகும், ஆனால் நீராவி வெளியேறாது)
*   **தனித்த அமைப்பு (Isolated System):** அமைப்பு சுற்றுப்புறங்களுடன் ஆற்றல் அல்லது பொருள் எதையும் பரிமாறிக்கொள்ள முடியாது.
    *   **உதாரணம்:** ஒரு தெர்மோஸ் குடுவையில் உள்ள சூடான தேநீர். (வெப்பமும் வெளியேறாது, பொருளும் வெளியேறாது - கொள்கையளவில்)

### 3. பண்புகளின் வகைகள் (Types of Properties)

*   **அகன்ற பண்புகள் (Extensive Properties):** அமைப்பில் உள்ள பொருளின் அளவைப் பொறுத்து அமையும் பண்புகள்.
    *   **உதாரணம்:** நிறை (mass), கனஅளவு (volume), ஆற்றல் (energy), என்தால்பி (enthalpy), என்ட்ரோபி (entropy). (தமிழில்: ஒரு பெரிய பாத்திரத்தில் அதிக தண்ணீர் இருக்கும், சிறிய பாத்திரத்தில் குறைவான தண்ணீர் இருக்கும் - அளவு சார்ந்தது)
*   **செறிந்த பண்புகள் (Intensive Properties):** அமைப்பில் உள்ள பொருளின் அளவைப் பொறுத்து அமையாத பண்புகள். இவை பொருளின் தன்மையைப் பொறுத்தது.
    *   **உதாரணம்:** வெப்பநிலை (temperature), அழுத்தம் (pressure), அடர்த்தி (density), பாகுத்தன்மை (viscosity), ஒளிவிலகல் எண் (refractive index). (தமிழில்: ஒரு பெரிய பாத்திரத்தில் உள்ள தண்ணீரும், சிறிய பாத்திரத்தில் உள்ள தண்ணீரும் ஒரே வெப்பநிலையில் இருக்கலாம் - அளவு சாராதது)

### 4. நிலைச் சார்புகள் மற்றும் வழிச் சார்புகள் (State Functions and Path Functions)

*   **நிலைச் சார்புகள் (State Functions):** ஒரு அமைப்பின் ஆரம்ப மற்றும் இறுதி நிலைகளை மட்டுமே சார்ந்துள்ள பண்புகள், அவை அடைந்த பாதையைச் சார்ந்திருக்காது.
    *   **உதாரணம்:** வெப்பநிலை (T), அழுத்தம் (P), கனஅளவு (V), அக ஆற்றல் (U), என்தால்பி (H), என்ட்ரோபி (S), கிப்ஸ் கட்டிலா ஆற்றல் (G). (தமிழில்: ஒரு மலையின் உச்சிக்கு நீங்கள் எந்தப் பாதையில் சென்றாலும், உங்கள் இறுதி உயரம் ஒரே மாதிரியாக இருக்கும் - அது ஒரு நிலைச் சார்பு)
*   **வழிச் சார்புகள் (Path Functions):** ஒரு அமைப்பு ஒரு நிலையிலிருந்து மற்றொரு நிலைக்கு மாறுவதற்கு எடுத்துக்கொண்ட பாதையைச் சார்ந்திருக்கும் பண்புகள்.
    *   **உதாரணம்:** வெப்பம் (q), வேலை (w). (தமிழில்: மலைக்குச் செல்ல நீங்கள் எந்தப் பாதையைத் தேர்வு செய்கிறீர்களோ, அதைப் பொறுத்து நீங்கள் கடக்கும் தூரம் (வேலை) மாறுபடும் - அது ஒரு வழிச் சார்பு)

### 5. வெப்ப இயக்கவியல் செயல்முறைகள் (Thermodynamic Processes)

*   **சமவெப்பச் செயல்முறை (Isothermal process):** வெப்பநிலை மாறாதது (ΔT = 0).
*   **சமஅழுத்தச் செயல்முறை (Isobaric process):** அழுத்தம் மாறாதது (ΔP = 0).
*   **சமபருமச் செயல்முறை (Isochoric process):** கனஅளவு மாறாதது (ΔV = 0).
*   **வெப்பமற்றச் செயல்முறை (Adiabatic process):** வெப்பம் பரிமாறப்படாது (q = 0).
*   **சுற்றுச் செயல்முறை (Cyclic process):** அமைப்பு ஒரு தொடர்ச்சியான மாற்றங்களுக்குப் பிறகு அதன் ஆரம்ப நிலைக்குத் திரும்புகிறது (ΔU = 0, ΔH = 0).

### 6. முதல் வெப்ப இயக்கவியல் விதி (First Law of Thermodynamics)

**ஆற்றல் அழிவின்மை விதி:** ஆற்றலை ஆக்கவோ அழிக்கவோ முடியாது, ஆனால் ஒரு வடிவத்திலிருந்து மற்றொரு வடிவத்திற்கு மாற்றலாம்.
*   **கணித வடிவம்:** `ΔU = q + w`
    *   `ΔU`: அமைப்பின் அக ஆற்றலில் ஏற்படும் மாற்றம்.
    *   `q`: அமைப்பு பரிமாறிக்கொள்ளும் வெப்பம். (+q: அமைப்பு வெப்பத்தை உறிஞ்சுகிறது, -q: அமைப்பு வெப்பத்தை வெளியிடுகிறது)
    *   `w`: அமைப்பு மீது அல்லது அமைப்பால் செய்யப்படும் வேலை. (+w: சுற்றுப்புறத்தால் அமைப்பு மீது வேலை செய்யப்படுகிறது, -w: அமைப்பால் சுற்றுப்புறத்தின் மீது வேலை செய்யப்படுகிறது)

*   **அழுத்த-கனஅளவு வேலை (Pressure-Volume Work):**
    *   வாயுவின் விரிவாக்கத்தால் செய்யப்படும் வேலை: `w = -P_ext * ΔV`
    *   வாயுவின் சுருக்கத்தால் செய்யப்படும் வேலை: `w = -P_ext * ΔV` (இங்கு ΔV எதிர்மறையாக இருக்கும், எனவே w நேர்மறையாக இருக்கும்)
    *   மீள் செயல்முறைக்கு: `w = -nRT ln(V2/V1)` அல்லது `w = w = -nRT ln(P1/P2)`

### 7. அக ஆற்றல் (Internal Energy, U) மற்றும் என்தால்பி (Enthalpy, H)

*   **அக ஆற்றல் (U):** ஒரு அமைப்பில் உள்ள அனைத்து வகையான ஆற்றல்களின் (இயக்க ஆற்றல், நிலை ஆற்றல்) கூட்டுத்தொகை. இது ஒரு நிலைச் சார்பு.
*   **என்தால்பி (H):** நிலையான அழுத்தத்தில் ஒரு அமைப்பின் மொத்த வெப்ப உள்ளடக்கம்.
    *   `H = U + PV`
    *   **என்தால்பி மாற்றம் (ΔH):** `ΔH = ΔU + PΔV` (நிலையான அழுத்தத்தில்)
    *   வாயு வினைகளுக்கு: `ΔH = ΔU + Δn_g RT` (Δn_g = வினை விளைபொருள் வாயு மோல்களின் எண்ணிக்கை - வினைபடு பொருள் வாயு மோல்களின் எண்ணிக்கை)

### 8. வெப்ப ஏற்புத்திறன் (Heat Capacity, C)

*   ஒரு பொருளின் வெப்பநிலையை 1 டிகிரி செல்சியஸ் அல்லது 1 கெல்வின் உயர்த்தத் தேவையான வெப்ப ஆற்றல்.
    *   `q = CΔT`
*   **நிலையான கனஅளவில் வெப்ப ஏற்புத்திறன் (Cv):** `Cv = (∂U/∂T)_V`
    *   `ΔU = n Cv ΔT`
*   **நிலையான அழுத்தத்தில் வெப்ப ஏற்புத்திறன் (Cp):** `Cp = (∂H/∂T)_P`
    *   `ΔH = n Cp ΔT`
*   **மேயர் சமன்பாடு:** `Cp - Cv = R` (ஒரு மோல் நல்லியல்பு வாயுவிற்கு)

### 9. என்தால்பி மாற்றங்கள் (Enthalpy Changes)

*   **வினை என்தால்பி (ΔH_reaction):** ஒரு குறிப்பிட்ட வெப்பநிலையில் மற்றும் அழுத்தத்தில் ஒரு வினை நிகழும்போது ஏற்படும் என்தால்பி மாற்றம்.
*   **தரம் என்தால்பி மாற்றம் (Standard Enthalpy Change, ΔH°):** 298 K மற்றும் 1 atm அழுத்தத்தில் என்தால்பி மாற்றம்.
*   **உருவாக்கத்தின் தரம் என்தால்பி (Standard Enthalpy of Formation, ΔH_f°):** ஒரு சேர்மத்தின் ஒரு மோல், அதன் தனிமங்களிலிருந்து அவற்றின் நிலையான வடிவங்களில் உருவாகும்போது ஏற்படும் என்தால்பி மாற்றம். (தனிமங்களின் நிலையான வடிவங்களுக்கு ΔH_f° = 0)
*   **எரிதலின் தரம் என்தால்பி (Standard Enthalpy of Combustion, ΔH_c°):** ஒரு சேர்மத்தின் ஒரு மோல் அதிகப்படியான ஆக்ஸிஜனில் முழுமையாக எரியும்போது ஏற்படும் என்தால்பி மாற்றம்.
*   **நடுநிலையாக்கலின் என்தால்பி (Enthalpy of Neutralization):** ஒரு கிராம் சமானம் அமிலம் ஒரு கிராம் சமானம் காரத்தால் நடுநிலையாக்கப்படும்போது ஏற்படும் என்தால்பி மாற்றம் (வலுவான அமிலம் மற்றும் காரத்திற்கு ~ -57.3 kJ/mol).
*   **பிணைப்பு என்தால்பி (Bond Enthalpy):** ஒரு மோல் பிணைப்புகளை வாயு நிலையில் உடைக்கத் தேவையான ஆற்றல்.
    *   `ΔH_reaction = Σ (வினைபடு பொருள்களின் பிணைப்பு என்தால்பிகள்) - Σ (வினைவிளை பொருள்களின் பிணைப்பு என்தால்பிகள்)`

### 10. ஹெஸ்ஸின் மாறா வெப்ப கூட்டுத்தொகை விதி (Hess's Law of Constant Heat Summation)

ஒரு வேதிவினை பல படிகளில் நடந்தாலும் அல்லது ஒரே படியில் நடந்தாலும், மொத்த என்தால்பி மாற்றம் ஒரே மாதிரியாக இருக்கும். (தமிழில்: நீங்கள் உங்கள் வீட்டிலிருந்து பள்ளிக்கு நேரடியாகச் சென்றாலும் அல்லது நண்பரின் வீட்டின் வழியாகச் சென்றாலும், உங்கள் வீடு மற்றும் பள்ளிக்கு இடையிலான தூர மாற்றம் (ΔH) ஒரே மாதிரியாக இருக்கும்.)

`ΔH_reaction = Σ ΔH_f° (வினைவிளைபொருள்கள்) - Σ ΔH_f° (வினைபடு பொருள்கள்)`

### 11. இரண்டாவது வெப்ப இயக்கவியல் விதி (Second Law of Thermodynamics)

*   **என்ட்ரோபி (Entropy, S):** ஒரு அமைப்பில் உள்ள ஒழுங்கின்மையின் அல்லது சீரற்ற தன்மையின் அளவு.
    *   `ΔS = q_rev / T` (மீள் செயல்முறைக்கு)
*   **தன்னிச்சையான செயல்முறைகள் (Spontaneous Processes):** ஒரு தன்னிச்சையான செயல்முறையின் போது, பிரபஞ்சத்தின் மொத்த என்ட்ரோபி அதிகரிக்கிறது.
    *   `ΔS_total = ΔS_system + ΔS_surroundings > 0` (தன்னிச்சையான செயல்முறைக்கு)
    *   `ΔS_total = 0` (சமநிலை செயல்முறைக்கு)
*   வாயு நிலை > திரவ நிலை > திண்ம நிலை: என்ட்ரோபி வரிசை.
*   மோல்களின் எண்ணிக்கை அதிகரிக்கும் போது என்ட்ரோபி அதிகரிக்கும்.
*   வெப்பநிலை அதிகரிக்கும் போது என்ட்ரோபி அதிகரிக்கும்.

### 12. மூன்றாவது வெப்ப இயக்கவியல் விதி (Third Law of Thermodynamics)

தனிச்சுழி வெப்பநிலையில் (0 கெல்வின்), ஒரு படிக திடப்பொருளின் என்ட்ரோபி பூஜ்ஜியமாக இருக்கும். (தமிழில்: ஒரு பொருள் உறைந்த தனிச்சுழி வெப்பநிலைக்கு வரும்போது, அதன் அனைத்து மூலக்கூறுகளும் ஒழுங்காக நிலைபெற்று, எந்த அசைவும் இல்லாமல் இருக்கும், எனவே ஒழுங்கின்மை பூஜ்ஜியமாகிறது.)

### 13. கிப்ஸ் கட்டிலா ஆற்றல் (Gibbs Free Energy, G)

*   ஒரு தன்னிச்சையான செயல்முறைக்குக் கிடைக்கும் அதிகபட்ச பயனுள்ள வேலை ஆற்றல்.
    *   `G = H - TS`
    *   **கிப்ஸ் கட்டிலா ஆற்றல் மாற்றம் (ΔG):** `ΔG = ΔH - TΔS`
*   **தன்னிச்சைக்கான நிபந்தனைகள் (Criteria for Spontaneity):**
    *   `ΔG < 0`: செயல்முறை தன்னிச்சையானது (Spontaneous).
    *   `ΔG > 0`: செயல்முறை தன்னிச்சையற்றது (Non-spontaneous).
    *   `ΔG = 0`: அமைப்பு சமநிலையில் உள்ளது (At equilibrium).

| ΔH | ΔS | ΔG = ΔH - TΔS | தன்னிச்சை (Spontaneity)                      |
|----|----|---------------|-----------------------------------------------|
| -  | +  | -             | அனைத்து வெப்பநிலையிலும் தன்னிச்சையானது      |
| +  | -  | +             | அனைத்து வெப்பநிலையிலும் தன்னிச்சையற்றது     |
| -  | -  | - (குறைந்த T) | குறைந்த வெப்பநிலையில் தன்னிச்சையானது           |
| -  | -  | + (அதிக T)    | அதிக வெப்பநிலையில் தன்னிச்சையற்றது          |
| +  | +  | + (குறைந்த T) | குறைந்த வெப்பநிலையில் தன்னிச்சையற்றது        |
| +  | +  | - (அதிக T)    | அதிக வெப்பநிலையில் தன்னிச்சையானது            |

### 14. ΔG மற்றும் சமநிலை மாறிலி (K) இடையே உள்ள தொடர்பு

*   `ΔG° = -RT ln K`
    *   `R`: வாயு மாறிலி (8.314 J K⁻¹ mol⁻¹)
    *   `T`: வெப்பநிலை (கெல்வினில்)
    *   `K`: சமநிலை மாறிலி
*   `ΔG = ΔG° + RT ln Q` (Q: வினைப் பகுதி)

## தொகுதி 2: தீர்க்கப்பட்ட எடுத்துக்காட்டுகள் (Worked Examples)

**எடுத்துக்காட்டு 1:** ஒரு அமைப்பு 20 J வெப்பத்தை உறிஞ்சி, 10 J வேலையைச் செய்கிறது. அமைப்பின் அக ஆற்றல் மாற்றத்தைக் கணக்கிடுக.
**தீர்வு:**
முதல் வெப்ப இயக்கவியல் விதியின்படி: `ΔU = q + w`
கொடுக்கப்பட்டவை:
`q = +20 J` (வெப்பத்தை உறிஞ்சுகிறது)
`w = -10 J` (அமைப்பு வேலை செய்கிறது)
`ΔU = 20 J + (-10 J) = 10 J`
அக ஆற்றல் மாற்றம் `+10 J`.

**எடுத்துக்காட்டு 2:** ஒரு வாயுவின் கனஅளவு 1 L இல் இருந்து 5 L ஆக மாறும்போது, 2 atm நிலையான வெளி அழுத்தத்திற்கு எதிராக விரிவடைகிறது. செய்யப்பட்ட வேலையைக் கணக்கிடுக.
**தீர்வு:**
செய்யப்பட்ட வேலை: `w = -P_ext * ΔV`
`P_ext = 2 atm`
`ΔV = V_final - V_initial = 5 L - 1 L = 4 L`
`w = -(2 atm) * (4 L) = -8 L atm`
`1 L atm = 101.3 J`
`w = -8 * 101.3 J = -810.4 J`

**எடுத்துக்காட்டு 3:** பின்வரும் தரவுகளைப் பயன்படுத்தி மீத்தேன் எரிதலின் தரம் என்தால்பியைக் கணக்கிடுக:
`ΔH_f°(CH₄(g)) = -74.8 kJ/mol`
`ΔH_f°(CO₂(g)) = -393.5 kJ/mol`
`ΔH_f°(H₂O(l)) = -285.8 kJ/mol`
**தீர்வு:**
எரிதல் வினை: `CH₄(g) + 2O₂(g) → CO₂(g) + 2H₂O(l)`
`ΔH_reaction° = [Σ ΔH_f° (வினைவிளைபொருள்கள்)] - [Σ ΔH_f° (வினைபடு பொருள்கள்)]`
`ΔH_reaction° = [ΔH_f°(CO₂(g)) + 2 * ΔH_f°(H₂O(l))] - [ΔH_f°(CH₄(g)) + 2 * ΔH_f°(O₂(g))]`
`O₂(g)` ஒரு தனிமம் அதன் நிலையான நிலையில், எனவே `ΔH_f°(O₂(g)) = 0`.
`ΔH_reaction° = [-393.5 + 2 * (-285.8)] - [-74.8 + 2 * 0]`
`ΔH_reaction° = [-393.5 - 571.6] - [-74.8]`
`ΔH_reaction° = -965.1 + 74.8 = -890.3 kJ/mol`

**எடுத்துக்காட்டு 4:** 298 K இல் ஒரு வினைக்கு `ΔH = +200 kJ/mol` மற்றும் `ΔS = +400 J/K mol` எனில், எந்த வெப்பநிலையில் வினை தன்னிச்சையாக மாறும்?
**தீர்வு:**
ஒரு வினை தன்னிச்சையாக இருக்க `ΔG < 0`.
சமநிலையில் `ΔG = 0`, அதாவது `ΔH - TΔS = 0`.
`T = ΔH / ΔS`
`ΔH = +200 kJ/mol = +200,000 J/mol`
`ΔS = +400 J/K mol`
`T = 200,000 J/mol / 400 J/K mol = 500 K`
`ΔH` நேர்மறை மற்றும் `ΔS` நேர்மறையாக இருப்பதால், அதிக வெப்பநிலைகளில் வினை தன்னிச்சையானது. எனவே, `T > 500 K` இல் வினை தன்னிச்சையாக மாறும்.

**எடுத்துக்காட்டு 5:** H-H பிணைப்பு ஆற்றல் 436 kJ/mol மற்றும் Cl-Cl பிணைப்பு ஆற்றல் 242 kJ/mol. HCl பிணைப்பு ஆற்றல் 431 kJ/mol எனில், `H₂(g) + Cl₂(g) → 2HCl(g)` வினைக்கான என்தால்பி மாற்றத்தைக் கணக்கிடுக.
**தீர்வு:**
`ΔH_reaction = Σ (வினைபடு பொருள்களின் பிணைப்பு என்தால்பிகள்) - Σ (வினைவிளை பொருள்களின் பிணைப்பு என்தால்பிகள்)`
`ΔH_reaction = [B.E.(H-H) + B.E.(Cl-Cl)] - [2 * B.E.(H-Cl)]`
`ΔH_reaction = [436 + 242] - [2 * 431]`
`ΔH_reaction = 678 - 862`
`ΔH_reaction = -184 kJ/mol`

**எடுத்துக்காட்டு 6:** ஒரு வாயு 25°C இல் 5 L முதல் 10 L வரை விரிவடைகிறது, 1 atm நிலையான வெளி அழுத்தத்திற்கு எதிராக. வெப்பம் 1200 J உறிஞ்சப்பட்டால், அக ஆற்றலில் ஏற்படும் மாற்றத்தைக் கணக்கிடுக.
**தீர்வு:**
`ΔU = q + w`
`q = +1200 J` (உறிஞ்சப்பட்டது)
`w = -P_ext * ΔV`
`P_ext = 1 atm`
`ΔV = 10 L - 5 L = 5 L`
`w = -1 atm * 5 L = -5 L atm`
`1 L atm = 101.3 J`
`w = -5 * 101.3 J = -506.5 J`
`ΔU = 1200 J + (-506.5 J) = 693.5 J`

**எடுத்துக்காட்டு 7:** 298 K இல், `2A(g) + B(g) ⇌ 3C(g)` என்ற வினைக்கு `ΔH° = -100 kJ` மற்றும் `ΔS° = -200 J/K` எனில், சமநிலை மாறிலி (K) இன் மதிப்பைக் கணக்கிடுக. (R = 8.314 J K⁻¹ mol⁻¹)
**தீர்வு:**
முதலில் `ΔG°` ஐக் கணக்கிடவும்:
`ΔG° = ΔH° - TΔS°`
`ΔH° = -100 kJ = -100,000 J`
`ΔS° = -200 J/K`
`T = 298 K`
`ΔG° = -100,000 J - (298 K * -200 J/K)`
`ΔG° = -100,000 J + 59,600 J`
`ΔG° = -40,400 J`

இப்போது `ΔG° = -RT ln K` சமன்பாட்டைப் பயன்படுத்தவும்:
`-40,400 J = -(8.314 J K⁻¹ mol⁻¹) * (298 K) * ln K`
`ln K = 40,400 / (8.314 * 298)`
`ln K = 40,400 / 2477.572`
`ln K ≈ 16.306`
`K = e^(16.306)`
`K ≈ 1.20 x 10⁷`

## தொகுதி 4: கூற்று-காரணம் கேள்விகள் (Assertion-Reason Questions)

**வழிமுறைகள்:** பின்வரும் கேள்விகளில், ஒரு கூற்றும் (Assertion) அதற்கு ஒரு காரணமும் (Reason) கொடுக்கப்பட்டுள்ளது. சரியான விருப்பத்தைத் தேர்ந்தெடுக்கவும்:
    a) கூற்று மற்றும் காரணம் இரண்டும் உண்மை, மேலும் காரணம் கூற்றுக்கு சரியான விளக்கமாகும்.
    b) கூற்று மற்றும் காரணம் இரண்டும் உண்மை, ஆனால் காரணம் கூற்றுக்கு சரியான விளக்கம் அல்ல.
    c) கூற்று உண்மை, ஆனால் காரணம் தவறு.
    d) கூற்று தவறு, ஆனால் காரணம் உண்மை.
    e) கூற்று மற்றும் காரணம் இரண்டும் தவறு.

**1. கூற்று (A):** ஒரு சுற்றோட்டச் செயல்முறையில் அக ஆற்றலில் ஏற்படும் மாற்றம் பூஜ்ஜியம்.
   **காரணம் (R):** அக ஆற்றல் ஒரு நிலைச் சார்பு.

**2. கூற்று (A):** ஒரு வெப்பமற்ற விரிவாக்கத்தின் போது, வாயுவின் வெப்பநிலை குறைகிறது.
   **காரணம் (R):** வெப்பமற்ற செயல்முறையில், அமைப்பால் செய்யப்பட்ட வேலை அக ஆற்றல் குறைவதற்கு காரணமாகிறது.

**3. கூற்று (A):** வெப்பம் மற்றும் வேலை இரண்டும் வழிச் சார்புகள்.
   **காரணம் (R):** அவை அமைப்பின் ஆரம்ப மற்றும் இறுதி நிலைகளை மட்டுமே சார்ந்துள்ளது.

**4. கூற்று (A):** அனைத்து தன்னிச்சையான செயல்முறைகளும் வெப்ப உமிழ்வு (exothermic) வினைகளாகும்.
   **காரணம் (R):** வெப்ப உமிழ்வு வினைகள் எப்போதும் தன்னிச்சையானவை.

**5. கூற்று (A):** தனித்த அமைப்பில் ஆற்றல் மற்றும் பொருள் பரிமாற்றம் இல்லை.
   **காரணம் (R):** அதன் எல்லைகள் வெப்பமற்றவை மற்றும் ஊடுருவ முடியாதவை.

**6. கூற்று (A):** `C_p` எப்போதும் `C_v` ஐ விட அதிகமாக இருக்கும்.
   **காரணம் (R):** நிலையான அழுத்தத்தில், ஒரு வாயு விரிவடைந்து வேலை செய்ய வேண்டும், இது நிலையான கனஅளவில் அதே வெப்பநிலை மாற்றத்திற்குத் தேவையானதை விட அதிக வெப்ப ஆற்றல் தேவைப்படுகிறது.

**7. கூற்று (A):** பிரபஞ்சத்தின் என்ட்ரோபி தொடர்ந்து அதிகரிக்கிறது.
   **காரணம் (R):** தன்னிச்சையான செயல்முறைகள் பிரபஞ்சத்தின் மொத்த என்ட்ரோபியை அதிகரிக்கின்றன.

**8. கூற்று (A):** `ΔG` இன் நேர்மறை மதிப்பு தன்னிச்சையற்ற செயல்முறையைக் குறிக்கிறது.
   **காரணம் (R):** `ΔG` என்பது ஒரு தன்னிச்சையான செயல்முறைக்கு கிடைக்கக்கூடிய பயனுள்ள வேலையின் அளவு.

**9. கூற்று (A):** `N₂(g) + 3H₂(g) → 2NH₃(g)` வினையின் போது என்ட்ரோபி குறைகிறது.
   **காரணம் (R):** வினை விளைபொருள் மோல்களின் எண்ணிக்கை வினைபடு பொருள் மோல்களின் எண்ணிக்கையை விட குறைவாக உள்ளது.

**10. கூற்று (A):** உருவாக்கத்தின் தரம் என்தால்பி (ΔH_f°) தனிமங்களுக்கு பூஜ்ஜியம்.
    **காரணம் (R):** தனிமங்கள் அவற்றின் நிலையான வடிவத்தில் வேதிவினைகளை உருவாக்கத் தேவையில்லை.

**11. கூற்று (A):** ஒரு தன்னிச்சையான வேதிவினை நிகழும்போது, `ΔG` இன் மதிப்பு எதிர்மறையாக இருக்கும்.
    **காரணம் (R):** `ΔG` மதிப்பு பூஜ்யமாக இருக்கும்போது ஒரு அமைப்பு சமநிலையில் இருக்கும்.

**12. கூற்று (A):** நீரின் ஆவியாதல் செயல்முறை வெப்ப உறிஞ்சி (endothermic) ஆகும்.
    **காரணம் (R):** திரவ நிலையிலிருந்து வாயு நிலைக்கு மாறும் போது மூலக்கூறுகளுக்கு இடையிலான பிணைப்புகளை உடைக்க ஆற்றல் தேவைப்படுகிறது.

**13. கூற்று (A):** திடப்பொருள்கள் திரவங்களை விட குறைந்த என்ட்ரோபியைக் கொண்டுள்ளன.
    **காரணம் (R):** திடப்பொருள்களில் மூலக்கூறுகள் மிக ஒழுங்கான முறையில் தொகுக்கப்பட்டு, அவற்றின் இயக்கம் கட்டுப்படுத்தப்படுகிறது.

**14. கூற்று (A):** ஒரு திறந்த அமைப்பு சுற்றுப்புறங்களுடன் ஆற்றல் மற்றும் பொருள் இரண்டையும் பரிமாறிக்கொள்ள முடியும்.
    **காரணம் (R):** அதன் எல்லைகள் வெப்பத்தை கடத்தும் மற்றும் ஊடுருவும் தன்மை கொண்டவை.

**15. கூற்று (A):** ஹெஸ்ஸின் விதி வெப்ப இயக்கவியல் முதல் விதியின் விளைவாகும்.
    **காரணம் (R):** என்தால்பி ஒரு நிலைச் சார்பு.
