# LinguaLeap

LinguaLeap is an AI-powered language learning assistant designed to help users master vocabulary and pronunciation through engaging exercises and personalized feedback.

## Features

-   **Smart Error Analysis**: usage of AI to analyze your pronunciation and grammar.
-   **Adaptive Learning Paths**: Personalized lessons based on your progress.
-   **Interactive Exercises**: Engaging ways to practice vocabulary and speaking.

## Getting Started

1.  Clone the repository:
    ```bash
    git clone https://github.com/bluepill02/lingualeap.git
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the development server:
    ```bash
    npm run dev
    ```

## Technolgies Used
- TypeScript
- React / Next.js
- TailwindCSS
- AI Integration
